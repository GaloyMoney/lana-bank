# Changelog

Release notes and version history for the Lana Banking Platform.

## Versioning

Lana follows semantic versioning. API changes are documented with each release.

- **Major versions** (X.0.0): Breaking changes
- **Minor versions** (x.Y.0): New features, backwards compatible
- **Patch versions** (x.y.Z): Bug fixes, backwards compatible
