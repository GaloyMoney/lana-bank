# Glosario

Términos clave utilizados en toda la documentación de Lana.

## Términos Bancarios

| Término | Definición |
|---------|------------|
| **Facilidad de Crédito** | Un acuerdo de préstamo pre-aprobado que permite pedir prestado hasta un límite especificado |
| **Desembolso** | La liberación de fondos del préstamo al prestatario |
| **Obligación** | Un pago programado que el prestatario debe cumplir |
| **Colateral** | Activos comprometidos como garantía de un préstamo |
| **KYC** | Know Your Customer (Conoce a Tu Cliente) - proceso de verificación de identidad |
| **KYB** | Know Your Business (Conoce a Tu Empresa) - proceso de verificación de entidades comerciales |

## Términos Técnicos

| Término | Definición |
|---------|------------|
| **Event Sourcing** | Patrón donde los cambios de estado se almacenan como una secuencia de eventos |
| **Patrón Outbox** | Asegura la publicación confiable de eventos junto con las transacciones de base de datos |
| **Arquitectura Hexagonal** | Patrón de diseño que separa la lógica central de las preocupaciones externas |
| **Contabilidad de Doble Entrada** | Cada transacción afecta al menos dos cuentas (débito y crédito) |

## Términos Específicos de Lana

| Término | Definición |
|---------|------------|
| **Cala** | El sistema de libro mayor subyacente que potencia la contabilidad de Lana |
| **API de Administración** | API GraphQL para operaciones internas/administrativas |
| **API de Cliente** | API GraphQL para aplicaciones orientadas al cliente |
| **Configuración de Dominio** | Sistema de configuración con tipado seguro para ajustes en tiempo de ejecución |

*Este glosario se expandirá a medida que crezca la documentación.*
