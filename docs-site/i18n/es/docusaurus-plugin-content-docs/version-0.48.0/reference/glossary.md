# Glosario

Términos clave utilizados en toda la documentación de Lana.

## Términos Bancarios

| Término | Definición |
|------|------------|
| **Facilidad de Crédito** | Un acuerdo de préstamo preaprobado que permite tomar prestado hasta un límite especificado |
| **Desembolso** | La liberación de los fondos del préstamo al prestatario |
| **Obligación** | Un pago programado que el prestatario debe cumplir |
| **Garantía** | Activos comprometidos como seguridad para un préstamo |
| **KYC** | Conozca a Su Cliente - proceso de verificación de identidad |
| **KYB** | Conozca a Su Empresa - proceso de verificación de entidad empresarial |

## Términos Técnicos

| Término | Definición |
|------|------------|
| **Event Sourcing** | Patrón donde los cambios de estado se almacenan como una secuencia de eventos |
| **Outbox Pattern** | Garantiza la publicación confiable de eventos junto con las transacciones de base de datos |
| **Hexagonal Architecture** | Patrón de diseño que separa la lógica central de las preocupaciones externas |
| **Contabilidad de Partida Doble** | Cada transacción afecta al menos dos cuentas (débito y crédito) |

## Términos Específicos de Lana

| Término | Definición |
|------|------------|
| **Cala** | El sistema de libro mayor subyacente que impulsa la contabilidad de Lana |
| **Admin API** | API GraphQL para operaciones internas/administrativas |
| **Customer API** | API GraphQL para aplicaciones orientadas al cliente |
| **Domain Config** | Sistema de configuración con tipos seguros para ajustes en tiempo de ejecución |

*Este glosario se ampliará a medida que crezca la documentación.*
