# Primeros Pasos con Lana

Esta guía te ayudará a ponerte en marcha con la Plataforma Bancaria Lana rápidamente.

## Elige tu Ruta

Dependiendo de tu rol, sigue la guía apropiada:

- **Personal del Banco**: [Documentación Técnica](../technical-documentation/) - Procesos de negocio y procedimientos del panel de administración
- **Desarrolladores Externos**: [Inicio Rápido para Desarrolladores Externos](../for-external-developers/quickstart) - Configura el acceso a la API y realiza tu primera llamada
- **Desarrolladores Internos**: [Guía para Desarrolladores Internos](../for-internal-developers/) - Configuración local y desarrollo del código base
- **Ingenieros de Plataforma**: [Guía de Plataforma](../for-platform-engineers/) - Arquitectura del sistema y despliegue

## Requisitos Previos

Antes de comenzar, asegúrate de tener:

- Credenciales de acceso para tu instancia de Lana
- Conocimiento de los conceptos básicos de GraphQL (para integración de API)
- Familiaridad con conceptos bancarios (líneas de crédito, contabilidad de libro mayor)

## Próximos Pasos

Una vez que estés configurado, explora las secciones de documentación relevantes para tu rol en la barra lateral.
