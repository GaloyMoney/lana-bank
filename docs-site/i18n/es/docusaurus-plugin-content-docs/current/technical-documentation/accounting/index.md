## Modelo Operativo

Las pantallas de contabilidad en el panel de administración exponen vistas del libro contable
basadas en partida doble (Cala). Operativamente se usan:
- **Plan de Cuentas** para estructura y detalle de cuentas.
- **Balanza de comprobación** para validar equilibrio contable.
- **Balance general** para posición financiera.
- **Estado de resultados** para desempeño del período.

## Recorrido en Panel de Administración: Configuración de Módulos

La configuración de módulos mapea depósitos y crédito a códigos padre del plan de cuentas.
Estos mapeos son críticos porque definen dónde se contabilizan las transacciones.
