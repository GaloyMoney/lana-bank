# Registro de Cambios

Notas de lanzamiento e historial de versiones de la Plataforma Bancaria Lana.

## Versionado

Lana sigue el versionado semántico. Los cambios de API se documentan con cada lanzamiento.

- **Versiones mayores** (X.0.0): Cambios incompatibles
- **Versiones menores** (x.Y.0): Nuevas funcionalidades, compatibles hacia atrás
- **Versiones de parche** (x.y.Z): Corrección de errores, compatibles hacia atrás

## Lanzamientos

Lanzamiento inicial de la documentación.
