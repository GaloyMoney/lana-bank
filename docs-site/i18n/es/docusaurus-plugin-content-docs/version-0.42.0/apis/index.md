# Referencia de APIs

Lana Bank expone dos APIs GraphQL y publica eventos de dominio para integración.

## APIs GraphQL

- **[Admin API](admin-api)** — API completa para operaciones de back-office incluyendo gestión de clientes, facilidades de crédito, depósitos, contabilidad y configuración del sistema.

- **[Customer API](customer-api)** — API orientada al cliente para acceso a cuentas, solicitudes de préstamos, operaciones de depósitos y gestión de documentos.

## Eventos de Dominio

- **[Eventos de Dominio](events)** — Eventos de dominio públicos publicados a través del patrón de outbox transaccional, disponibles para integración, analítica y auditoría.
