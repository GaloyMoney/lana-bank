# Comenzando con Lana

Esta guía te ayudará a empezar rápidamente con la Plataforma Bancaria Lana.

## Elige Tu Camino

Dependiendo de tu rol, sigue la guía apropiada:

- **Personal del Banco**: [Documentación Técnica](../technical-documentation/) - Procesos de negocio y procedimientos del panel de administración
- **Desarrolladores Externos**: [Inicio Rápido para Desarrolladores Externos](../for-external-developers/quickstart) - Configura el acceso a la API y haz tu primera llamada
- **Desarrolladores Internos**: [Guía del Desarrollador Interno](../for-internal-developers/) - Configuración local y desarrollo del código base
- **Ingenieros de Plataforma**: [Guía de Plataforma](../for-platform-engineers/) - Arquitectura del sistema y despliegue

## Prerrequisitos

Antes de comenzar, asegúrate de tener:

- Credenciales de acceso para tu instancia de Lana
- Comprensión básica de GraphQL (para integración con API)
- Familiaridad con conceptos bancarios (facilidades de crédito, contabilidad de libro mayor)

## Siguientes Pasos

Una vez configurado, explora las secciones de documentación relevantes a tu rol en la barra lateral.
