# Documentación de Lana Bank

Bienvenido a la documentación técnica de Lana Bank.

Esta documentación cubre los componentes principales del sistema Lana Bank:

- **Lana Core**: El sistema bancario principal con diagramas de relación de entidades
- **Cala Ledger**: El sistema de libro mayor subyacente para el procesamiento de transacciones
- **Sistema de Crédito**: Documentación para facilidades de crédito y procesos relacionados

## Comenzando

Navegue a través de la documentación usando el menú lateral o los enlaces anteriores.

## Descripción General de la Arquitectura

Lana Bank sigue un patrón de arquitectura hexagonal con una clara separación entre:
- Lógica de dominio central
- Servicios de aplicación
- Adaptadores de infraestructura

Para especificaciones técnicas detalladas, consulte la documentación de cada componente individual.
