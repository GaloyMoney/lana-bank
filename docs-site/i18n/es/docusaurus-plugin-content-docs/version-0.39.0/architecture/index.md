# Arquitectura

Esta sección contiene documentación completa sobre la arquitectura técnica y funcional de Lana Bank.

## Arquitectura Funcional

El documento de [Arquitectura Funcional](./functional-architecture.md) proporciona una visión completa de:

- **Arquitectura de la Aplicación**: Módulos del core bancario, capa de aplicación y frontends web
- **Flujos de Comunicación**: Event sourcing, patrón outbox, jobs asíncronos y webhooks
- **Integraciones Externas**: KYC/KYB, pasarelas de pago, sistemas regulatorios y observabilidad
- **Seguridad**: Autenticación, autorización, cifrado, firewalls y VPN
- **Infraestructura**: Segmentación de red, zonas de seguridad, servidores, bases de datos y middleware

Esta documentación está destinada a equipos técnicos que necesitan comprender el diseño del sistema, los requisitos de despliegue y las características operativas.
