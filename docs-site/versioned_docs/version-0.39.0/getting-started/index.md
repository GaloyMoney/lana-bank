# Getting Started with Lana

This guide will help you get up and running with Lana Banking Platform quickly.

## Choose Your Path

Depending on your role, follow the appropriate guide:

- **Bank Staff**: [Technical Documentation](../technical-documentation/) - Business processes and admin panel procedures
- **External Developers**: [External Developer Quickstart](../for-external-developers/quickstart) - Set up API access and make your first call
- **Internal Developers**: [Internal Developer Guide](../for-internal-developers/) - Local setup and codebase development
- **Platform Engineers**: [Platform Guide](../for-platform-engineers/) - System architecture and deployment

## Prerequisites

Before you begin, ensure you have:

- Access credentials for your Lana instance
- Understanding of GraphQL basics (for API integration)
- Familiarity with banking concepts (credit facilities, ledger accounting)

## Next Steps

Once you're set up, explore the documentation sections relevant to your role in the sidebar.
