# Architecture

This section contains comprehensive documentation about the technical and functional architecture of Lana Bank.

## Functional Architecture

The [Functional Architecture](./functional-architecture.md) document provides a complete overview of:

- **Application Architecture**: Banking core modules, application layer, and web frontends
- **Communication Flows**: Event sourcing, outbox pattern, asynchronous jobs, and webhooks
- **External Integrations**: KYC/KYB, payment gateways, regulatory systems, and observability
- **Security**: Authentication, authorization, encryption, firewalls, and VPN
- **Infrastructure**: Network segmentation, security zones, servers, databases, and middleware

This documentation is intended for technical teams who need to understand the system's design, deployment requirements, and operational characteristics.
