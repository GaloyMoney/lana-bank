# API Reference

Lana Bank exposes two GraphQL APIs and publishes domain events for integration.

## GraphQL APIs

- **[Admin API](admin-api/api-reference.mdx)** — Full-featured API for back-office operations including customer management, credit facilities, deposits, accounting, and system configuration.

- **[Customer API](customer-api/api-reference.mdx)** — Customer-facing API for account access, loan applications, deposit operations, and document management.

## Domain Events

- **[Domain Events](events/events.md)** — Public domain events published via the transactional outbox pattern, available for integration, analytics, and audit purposes.
