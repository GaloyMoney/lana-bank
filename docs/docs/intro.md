# Introduction

Welcome to the Lana Bank documentation.

## Getting Started

Lana Bank is an enterprise banking platform built with event-sourcing and hexagonal architecture.

### Key Features

- **Credit Facilities** - Manage credit facilities, disbursals, and repayments
- **Accounting** - Full double-entry accounting with fiscal year support
- **GraphQL APIs** - Admin and Customer APIs for integration

## Architecture

The platform follows Domain-Driven Design (DDD) principles with:

- Event sourcing for audit trails
- Hexagonal architecture for clean separation of concerns
- GraphQL for flexible API queries
